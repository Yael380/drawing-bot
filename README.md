# DrawingBot

注专转 住住转 React 志.NET Core 爪专转 爪专 驻 驻拽转 砖驻 注转,  砖专转 爪专, 爪转 转转 拽转 砖 砖转砖, 砖专转 住专转 驻注转 专转 爪专 拽抓. 注专转 转 专爪 注  砖驻 爪专 转专 驻专驻 驻拽转 爪专.

##  转

- **Frontend**: React, TypeScript, React Hook Form, CSS 转, Canvas API  
- **Backend**: ASP.NET Core Web API  
- **DB Models**: `User`, `Image`, `Drawing`  
- **砖驻转 转转**: TypeScript, C#  
- **住**: 砖专转 拽 + 砖驻转 拽爪 砖专转

##  驻爪'专 注拽专

- 转转 砖 注   驻拽转 砖驻 注转 专 爪专 转  
- 爪专 注  拽住 驻 驻拽转 转专转 (砖: "爪专 转")  
- 转 志4 住 爪专转 : `circle`, `rectangle`, `triangle`, `line`  
- 转爪转 专 砖 转转 砖砖专 注  砖转砖  
- 驻砖专转 注 砖 转 拽转 爪专 注  
- 砖 专 专转 转 驻 转转 爪专  
- **驻砖专转 专转 爪专 拽抓 转 (PNG)**  
- **砖专转 住专转 驻拽转 爪专  爪专**

##  拽转 爪专

-  驻专驻 转专  JSON  砖 爪专转  
- 专拽 爪专 专砖 驻专 住驻爪驻 爪注 ( 住爪 砖)  

##  转拽 专爪

```bash
git clone https://github.com/Yael380/Drawing-bot
cd Drawing-bot

# 转拽转 爪 拽
